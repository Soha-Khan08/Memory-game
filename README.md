# Memory-game
This Java card‑matching game repeatedly asks the player to select two cards, reveals them temporarily, and hides non‑matching pairs while tracking elapsed time and move count. A win message appears once all pairs are matched, followed by historical records of time and moves. Input errors are caught and handled gracefully. The system checks selections for matches, keeps matched cards face up or removes them, and detects when the game is won. Undo functionality is supported through structured state management, and JUnit tests verify win detection. The code follows good style with clear comments, variable names, and modular methods.
